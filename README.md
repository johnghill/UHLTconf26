# About
This repo provides a resource for educators who wish to introduce undergraduate students to local language modules (LLMs). The resource is suitable for second year undergraduates, and was prepared for business students.

# How to obtain?
### Git CLI
`git clone https://github.com/johnghill/UHLTconf26`
### Manual Downloads
1. Download `.zip` from the menus
2. Download any individual file displayed in the grid above.

# How to use?
### Markdown
Markdown source `.md` may be compiled using the [marp-vscode](https://github.com/marp-team/marp-vscode) plugin for Visual Studio Code.
### Compiled formats
Download PowerPoint or PDF as normal.
